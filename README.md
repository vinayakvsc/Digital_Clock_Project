# ⏰ Digital Clock

This is a simple **Digital Clock web project** that displays the current time and date in real time.
I built this project while practicing **JavaScript, HTML, and CSS**, focusing on DOM manipulation and working with the JavaScript `Date` object.

The clock updates automatically every second and shows the time in **12-hour format with AM/PM**.

---

## ✨ Features

* Live digital clock that updates every second
* 12-hour time format with **AM / PM**
* Displays the **current date**
* Clean glass-style clock design
* Beginner-friendly and easy to understand code

---

## 🛠 Technologies Used

* **HTML** – Structure of the page
* **CSS** – Styling and layout
* **JavaScript** – Logic for updating time and date

## ⚙️ How It Works

The project uses JavaScript's **Date object** to get the current time.

Every second:

* The script gets the current hours, minutes, and seconds
* Converts the time to **12-hour format**
* Updates the clock on the page
* Displays the current date

This update happens automatically using **setInterval()**.
